🚀 InnerSpace AI Chat Application

A full-stack AI-powered chat application with authentication, persistent chat history, and real-time responses — built using Flask and Groq API.

🌐 Live Demo

🔗 https://innerspace-ai.onrender.com



✨ Features
🔐 User Authentication (Signup/Login/Logout)
💬 Real-time AI Chat
💾 Persistent Chat History (SQLite)
🤖 AI Integration using Groq API
⚡ Fast and responsive UI
🌍 Deployed on Render


🛠️ Tech Stack
Technology	Usage
Python	Backend logic
Flask	Web framework
SQLite	Database
HTML/CSS/JS	Frontend
Groq API	AI responses
Render	Deployment

⚙️ Installation (Run Locally)

1️⃣ Clone the repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name


2️⃣ Create virtual environment
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows


3️⃣ Install dependencies
pip install -r requirements.txt


4️⃣ Set environment variables

Create a .env file:
GROQ_API_KEY=your_api_key_here


5️⃣ Run the app
python app.py
🚀 Deployment

The app is deployed using Render.

Key configurations:
Python version: 3.10
Start command:
gunicorn app:app
